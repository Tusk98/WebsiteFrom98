AutoSort

AutoSort began as a friend name Martin Zou complaining about having to constantly move downloaded MP3 files from the downloads folder to his desktop. It became inspiration for me to create this desktop application created using Java and Swing to move MP3 files. After completion and distributing to friends, by popular demand I added in the functionality to move Microsoft Word documents as well as PDF files.
